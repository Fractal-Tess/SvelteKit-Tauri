## **Usage**

---

#### **Clone the repo**

```bash
git clone https://github.com/Fractal-Tess/Svelte-Kit-Tauri-Template
```

#### **Install dependencies**

```bash
npm install
```

### **Run the development server**

```bash
npm run tauri dev
```

### **Build the application**

```bash
npm run tauri build
```
